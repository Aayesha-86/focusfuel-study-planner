# FocusFuel

FocusFuel is a **smart study & energy planner** web app designed to help students manage focus, energy, and productivity.  

## Features
- Add study/focus sessions with subject, time, energy level, mood, and priority
- Smart suggestions based on energy levels (e.g., take a break if low)
- 25-minute focus timer with 5-minute breaks
- Mark sessions as completed
- Daily progress percentage tracker
- Modern, clean, mobile-friendly UI
- Data saved in browser storage (no backend required)

## Built With
- React
- Vite
- Local Storage
- Modern UI/UX principles



## Why FocusFuel?
Students often burn out while studying. FocusFuel balances focus, energy, and breaks to boost productivity.
